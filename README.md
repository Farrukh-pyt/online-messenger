# Online Messenger

This is a simple online messenger built with Python and Flask.

I created this project while learning web development and experimenting with basic chat system logic.

## Features

- Send messages
- View chat history
- Simple web interface
- Multiple users can open the chat

## Technologies Used

- Python
- Flask
- HTML/CSS

## How to run

1. Install Flask
2. Run:

python app.py

3. Open:

http://127.0.0.1:5000
